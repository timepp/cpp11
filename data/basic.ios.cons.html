[basic.ios.cons]27.5.5.2 basic_ios constructors

explicit basic_ios(basic_streambuf<charT,traits>* sb);

1Effects: Constructs an object of class basic_ios, assigning initial values to its member objects by calling init(sb).

basic_ios();

2Effects: Constructs an object of class basic_ios (27.5.3.7) leaving its member objects uninitialized. The object shall be initialized by calling basic_ios::init before its first use or before it is destroyed, whichever comes first; otherwise the behavior is undefined.

~basic_ios();

3Remarks: The destructor does not destroy rdbuf().

void init(basic_streambuf<charT,traits>* sb);
