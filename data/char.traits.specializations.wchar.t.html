  template<> struct char_traits<wchar_t> {
    typedef wchar_t     char_type;
    typedef wint_t      int_type;
    typedef streamoff   off_type;

    static void assign(char_type& c1, const char_type& c2) noexcept;

    static constexpr bool lt(char_type c1, char_type c2) noexcept;

    static char_type* copy(char_type* s1, const char_type* s2, size_t n);

1The defined types for int_type, pos_type, and state_type shall be wint_t, wstreampos, and mbstate_t respectively.

2The type wstreampos shall be an implementation-defined type that satisfies the requirements for pos_type in 27.2.2 and 27.3.

3The type mbstate_t is defined in <cwchar> and can represent any of the conversion states that can occur in an implementation-defined set of supported multibyte character encoding rules.

4The two-argument members assign, eq, and lt shall be defined identically to the built-in operators =, ==, and < respectively.
