[func.wrap.func.inv]20.8.11.2.4 function invocation
R operator()(ArgTypes... args) const;

1Effects: INVOKE(f, std::forward<ArgTypes>(args)..., R) (20.8.2), where f is the target object (20.8.1) of *this.

2Returns: Nothing if R is void, otherwise the return value of INVOKE(f, std::forward<ArgTypes>(args)..., R).

3Throws: bad_function_call if !*this; otherwise, any exception thrown by the wrapped callable object.
