[func.wrap.func.targ]20.8.11.2.5 function target access
const std::type_info& target_type() const noexcept;

1Returns: If *this has a target of type T, typeid(T); otherwise, typeid(void).

template<typename T> T* target() noexcept; template<typename T> const T* target() const noexcept;

2Requires: T shall be a type that is Callable (20.8.11.2) for parameter types ArgTypes and return type R.

3Returns: If target_type() == typeid(T) a pointer to the stored function target; otherwise a null pointer.
