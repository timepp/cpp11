[gram.key]A.1 Keywords

1New context-dependent keywords are introduced into a program by typedef (7.1.3), namespace (7.3.1), class (Clause 9), enumeration (7.2), and template (Clause 14) declarations.

identifier

original-namespace-name

class-name:
identifier

Note that a typedef-name naming a class is also a class-name (9.1).
