[input.output]27 Input/output library

[input.output.general]27.1 General

1This Clause describes components that C++ programs may use to perform input/output operations.

2The following subclauses describe requirements for stream parameters, and components for forward declarations of iostreams, predefined iostreams objects, base iostreams classes, stream buffering, stream formatting and manipulators, string streams, and file streams, as summarized in Table 121.
