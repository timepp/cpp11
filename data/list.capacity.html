[list.capacity]23.3.5.3 list capacity

void resize(size_type sz);

1Effects: If sz <= size(), equivalent to list<T>::iterator it = begin(); advance(it, sz); erase(it, end());. If size() < sz, appends sz - size() value-initialized elements to the sequence.

2Requires: T shall be DefaultConstructible.

void resize(size_type sz, const T& c);

3Effects: 

if (sz > size())
  insert(end(), sz-size(), c);
else if (sz < size()) {
  iterator i = begin();
  advance(i, sz);
  erase(i, end());
}
else
  ;                 // do nothing

4Requires: T shall be CopyInsertable into *this.
