[meta.unary.cat]20.9.4.1 Primary type categories

1The primary type categories correspond to the descriptions given in section 3.9 of the C++ standard.

2For any given type T, the result of applying one of these templates to T and to cv-qualified T shall yield the same result.

3[ Note: For any given type T, exactly one of the primary type categories has a value member that evaluates to true. — end note ]
