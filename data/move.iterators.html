[move.iterators]24.5.3 Move iterators

1Class template move_iterator is an iterator adaptor with the same behavior as the underlying iterator except that its dereference operator implicitly converts the value returned by the underlying iterator’s dereference operator to an rvalue reference. Some generic algorithms can be called with move iterators to replace copying with moving.

2[ Example: 

list<string> s;
// populate the list s
vector<string> v1(s.begin(), s.end());              // copies strings into v1
vector<string> v2(make_move_iterator(s.begin()),
                  make_move_iterator(s.end()));     // moves strings into v2

— end example ]
