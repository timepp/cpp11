[narrow.stream.objects]27.4.2 Narrow stream objects

istream cin;

1The object cin controls input from a stream buffer associated with the object stdin, declared in <cstdio>.

2After the object cin is initialized, cin.tie() returns &cout. Its state is otherwise the same as required for basic_ios<char>::init (27.5.5.2).

ostream cout;

3The object cout controls output to a stream buffer associated with the object stdout, declared in <cstdio> (27.9.2).

ostream cerr;

4The object cerr controls output to a stream buffer associated with the object stderr, declared in <cstdio> (27.9.2).

5After the object cerr is initialized, cerr.flags() & unitbuf is nonzero and cerr.tie() returns &cout. Its state is otherwise the same as required for basic_ios<char>::init (27.5.5.2).

ostream clog;

6The object clog controls output to a stream buffer associated with the object stderr, declared in <cstdio> (27.9.2).
