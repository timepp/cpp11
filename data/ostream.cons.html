[ostream.cons]27.7.3.2 basic_ostream constructors

explicit basic_ostream(basic_streambuf<charT,traits>* sb);

1Effects: Constructs an object of class basic_ostream, assigning initial values to the base class by calling basic_ios<charT,traits>::init(sb) (27.5.5.2).

2Postconditions: rdbuf() == sb.

virtual ~basic_ostream();

3Effects: Destroys an object of class basic_ostream.

4Remarks: Does not perform any operations on rdbuf().

basic_ostream(basic_ostream&& rhs);

5Effects: Move constructs from the rvalue rhs. This is accomplished by default constructing the base class and calling basic_ios<charT, traits>::move(rhs) to initialize the base class.
