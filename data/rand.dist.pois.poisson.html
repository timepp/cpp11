[rand.dist.pois]26.5.8.4 Poisson distributions

[rand.dist.pois.poisson]26.5.8.4.1 Class template poisson_distribution

1A poisson_distribution random number distribution produces integer values i ≤ 0 distributed according to the discrete probability function P(i | μ) = e−μμi / i!. The distribution parameter μ is also known as this distribution’s mean.

template<class IntType = int>
  class poisson_distribution
{
public:
  // types
  typedef IntType result_type;
  typedef unspecified param_type;

  // constructors and reset functions
  explicit poisson_distribution(double mean = 1.0);
  explicit poisson_distribution(const param_type& parm);
  void reset();

  // generating functions
  template<class URNG>
    result_type operator()(URNG& g);
  template<class URNG>
    result_type operator()(URNG& g, const param_type& parm);

  // property functions

  param_type param() const;
  void param(const param_type& parm);
  result_type min() const;
  result_type max() const;
};
explicit poisson_distribution(double mean = 1.0);

3Effects: Constructs a poisson_distribution object; mean corresponds to the parameter of the distribution.

double mean() const;
