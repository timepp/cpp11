[runtime.error]19.2.6 Class runtime_error 

namespace std {
  class runtime_error : public exception {
  public:
    explicit runtime_error(const string& what_arg);

  };
}

1The class runtime_error defines the type of objects thrown as exceptions to report errors presumably detectable only when the program executes.

runtime_error(const string& what_arg);

2Effects: Constructs an object of class runtime_error.

3Postconditions: strcmp(what(), what_arg.c_str()) == 0.
