[stmt.block]6.3 Compound statement or block

1So that several statements can be used where one is expected, the compound statement (also, and equivalently, called “block”) is provided.

compound-statement:
{ statement-seqopt }
statement-seq:
statement
statement-seq statement

A compound statement defines a block scope (3.3). [ Note: A declaration is a statement (6.7). — end note ]
