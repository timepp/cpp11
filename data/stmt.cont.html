[stmt.cont]6.6.2 The continue statement

1The continue statement shall occur only in an iteration-statement and causes control to pass to the loop-continuation portion of the smallest enclosing iteration-statement, that is, to the end of the loop. More precisely, in each of the statements

while (foo) {
  {
    // …
  }
contin: ;
}
do {
  {
    // …
  }
contin: ;
} while (foo);
for (;;) {
  {
    // …
  }
contin: ;
}

a continue not contained in an enclosed iteration statement is equivalent to goto contin.
