[stmt.if]6.4.1 The if statement

1If the condition (6.4) yields true the first substatement is executed. If the else part of the selection statement is present and the condition yields false, the second substatement is executed. In the second form of if statement (the one including else), if the first substatement is also an if statement then that inner if statement shall contain an else part.88

88) In other words, the else is associated with the nearest un-elsed if.
