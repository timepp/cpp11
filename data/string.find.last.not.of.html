size_type find_last_not_of(const basic_string& str, size_type pos = npos) const noexcept;

1Effects: Determines the highest position xpos, if possible, such that both of the following conditions obtain:

  • xpos <= pos and xpos < size();
  • traits::eq(at(xpos), str.at(I)) for no element I of the string controlled by str.

2Returns: xpos if the function can determine such a value for xpos. Otherwise, returns npos.

3Remarks: Uses traits::eq().
