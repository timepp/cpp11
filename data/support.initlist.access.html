[support.initlist.access]18.9.2 Initializer list access

const E* begin() const noexcept;

1Returns: A pointer to the beginning of the array. If size() == 0 the values of begin() and end() are unspecified but they shall be identical.

const E* end() const noexcept;

2Returns: begin() + size()

size_t size() const noexcept;

3Returns: The number of elements in the array.

4Complexity: constant time.
