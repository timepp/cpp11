[temp.arg.type]14.3.1 Template type arguments

1A template-argument for a template-parameter which is a type shall be a type-id.

2[ Example: 

template <class T> class X { };
template <class T> void f(T t) { }
struct { } unnamed_obj;

void f() {
  struct A { };
  enum { e1 };
  typedef struct { } B;
  B b;
  X<A> x1;          // OK
  X<A*> x2;         // OK
  X<B> x3;          // OK
  f(e1);            // OK
  f(unnamed_obj);   // OK
  f(b);             // OK
}

— end example ]

[ Note: A template type argument may be an incomplete type (3.9). — end note ]

3If a declaration acquires a function type through a type dependent on a template-parameter and this causes a declaration that does not use the syntactic form of a function declarator to have function type, the program is ill-formed.

[ Example: 

template<class T> struct A {
  static T t;
};
typedef int function();
A<function> a;          // ill-formed: would declare A<function>::t
                        // as a static member function

— end example ]
