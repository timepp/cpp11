[temp.mem.enum]14.5.1.4 Enumeration members of class templates

1An enumeration member of a class template may be defined outside the class template definition.

[ Example: 

template<class T> struct A {
  enum E : T;
};
A<int> a;
template<class T> enum A<T>::E : T { e1, e2 };
A<int>::E e = A<int>::e1;

— end example ]
