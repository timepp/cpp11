[exception.terminate]18.8.3 Abnormal termination

[terminate.handler]18.8.3.1 Type terminate_handler 

typedef void (*terminate_handler)();

1The type of a handler function to be called by terminate() when terminating exception processing.

2Required behavior: A terminate_handler shall terminate execution of the program without returning to the caller.

3Default behavior: The implementation’s default terminate_handler calls abort().
