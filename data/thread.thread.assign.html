[thread.thread.assign]30.3.1.4 thread assignment

thread& operator=(thread&& x) noexcept;

1Effects: If joinable(), calls terminate(). Otherwise, assigns the state of x to *this and sets x to a default constructed state.

2Postconditions: x.get_id() == id() and get_id() returns the value of x.get_id() prior to the assignment.
