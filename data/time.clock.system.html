[time.clock.system]20.11.7.1 Class system_clock

1Objects of class system_clock represent wall clock time from the system-wide realtime clock.

class system_clock {
public:
  typedef see below                         rep;
  typedef ratio<unspecified, unspecified>   period;
  typedef chrono::duration<rep, period>     duration;
  typedef chrono::time_point<system_clock>  time_point;
  static const bool is_steady =             unspecified;

  static time_point now() noexcept;

  // Map to C API
  static time_t     to_time_t  (const time_point& t) noexcept;
  static time_point from_time_t(time_t t) noexcept;
};
typedef unspecified system_clock::rep;

2Requires: system_clock::duration::min() < system_clock::duration::zero() shall be true. [ Note: This implies that rep is a signed type. — end note ]

static time_t to_time_t(const time_point& t) noexcept;

3Returns: A time_t object that represents the same point in time as t when both values are restricted to the coarser of the precisions of time_t and time_point. It is implementation defined whether values are rounded or truncated to the required precision.

static time_point from_time_t(time_t t) noexcept;

4Returns: A time_point object that represents the same point in time as t when both values are truncated restricted to the coarser of the precisions of time_t and time_point. It is implementation-defined whether values are rounded or truncated to the required precision.
